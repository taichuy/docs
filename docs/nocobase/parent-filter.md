# 父级筛选传递值插件

>解决场景：页面中，在父级链接区块之后，添加时候无法获取父级筛选区块值。
>
>问题：比如说通过表格区块链接其他区块，但是新增时候发现无法添加上级记录
>
>解决：使用该插件之后，就可以直接根据使用父级区块上级值了



该功能由插件：``@taichuy/plugin-block-filters-value``提供
